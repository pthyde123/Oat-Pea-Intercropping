Overview of general steps

  1. Create seedlot

  2. Pick accessions to plant for a new trial

  3. Join new trial accessions to seedlot accessions

  4. Pull bulk seed packs

  5. Pack plot envelops

  6. Subtract seed weight from seedlot



1. Create seedlots from an existing trial

Seedlots are created in T3 by transferring ‘Grain Weight g’ plot data to seed lots. When creating the seedlot a template is used for naming the seedlot. Have the source plot number identified so that the harvest bags can be easily located.


This format work well.

{accession_name}-{trial_name}-PLOT_{plot_number}
  • If the trial name is long it becomes a problem printing the seedlot name on envelopes or labels. In the future it might be best to keep the trial names shorter or use and abbreviation when creating the seedlot names. {accession_name}-trial_abbr.-PLOT_{plot_number}


Sources for seed lots can be accession, plot, or subplot

  • Functionally, we source the seed by plot and greatest seed weight for each given accession. These seed bags are separated out and used to fill seed packs. That way we are dealing with fewer bags. For example, the 2024 winter trial had 378 plots but 94 accession. We work with 94 bags, have the in a separate box or tray organized by plot number. This allows for easy access.
  • Sub-setting the seedlot sorce by management factor looks to be available but doesn’t seem to work, it freezes up. This could be useful if we wanted to select only mono-crop plots for seed purity purposes.


The following variables can also be used in the template

| trial_name

| accession_name

| plot_name

| plot_number

| rep block

| index - an incrementing index number, starting with 1
